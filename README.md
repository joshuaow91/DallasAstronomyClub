# How It's Made:
## Tech used: HTML, CSS & JavaScript

Welcome to my Github project! In this repository, you will find a website that showcases NASA's Picture of the Day using the NASA API. The website has been developed using JavaScript, HTML, and CSS to provide an aesthetically pleasing interface that is user-friendly and intuitive. This project has been developed with a focus on providing an easy way to access and view the stunning and awe-inspiring images from NASA. The API provides a wide range of pictures and information that can be used in various ways, and this project demonstrates a creative way to display them. Whether you are an astronomy enthusiast or just looking for some inspiration, this website is a great resource to explore. So go ahead and check out my Github repository to see how I have used the NASA API to create a visually stunning website.



## Optimizations
 - Implement a local storage feature for users to see previously searched media.
 - Utilize responsive design to ensure that the website looks and functions optimally on all devices, including desktops, tablets, and mobile phones.
